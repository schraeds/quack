setAutoSize(w,h)
Enables/Disables automatic resizing of either the width and/or height of a layer.

example:
var lyr = new DynLayer('<table><tr><td>Hello <br> World</td></tr></table>');
lyr.setAutoSize(true,true);
dynapi.functions.addChild(lyr);
