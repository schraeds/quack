generateBlueprint(type) - DynLayerInline
type - (String) Optional values:  css or js

When type is set to 'css' the element will generate a style sheet for the layer. When it's set to 'js' the element will generate the <div> structure in JavaScript code. If it's set to null only the plain <div> structure will be generated.
