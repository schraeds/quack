Known Issues

  • KB001 - Whenever an <ilayer> containing an XImage (dynapi.functions.getImage()) is placed inside a <table> the rollover effects will not work. This seems to be a bug in NS4
     
  • KB002 - When using TemplateManager Mozilla will collapse/expand table cells whenever the content of the layer changes. To solve this, specify a width for your table cells
     
  • KB003 - Molliza 1.4 generates a Pemission Denied error whenever the MouseEvent Class is included. This is caused by bug inside the Molliza Browser. The Molliza group is aware of this bug. This problem does NOT exist in Molliza 1.0 to 1.3
