dynapi.fx.TimerX - Quick Reference

Extends: DynLayer

Remarks
The TimerX Class is a simple extension to the DynLayer Class which is used to trigger timer events at set intervals.

Example: 
myLayer=new DynLayer(null,135,70,100,50);
myLayer.setBgColor('#000000');

var e={};
e.ontimer=function(e){
   var o=e.getSource();
   var src='<font size=2 color=#ffffff>';
   src+='o.getTickCount()+'</font>';
   o.setHTML(src);
}
myLayer.addEventListener(e);
myLayer.startTimer(1000);
dynapi.document.addChild(myLayer);

Public Methods

startTimer( interval) - Enable timer and set the interval 
interval - (int) Milliseconds between calls
stopTimer() - Disable timer

setTimerInterval(interval) Sets the number of milliseconds between calls to the timer event 

interval - (int) Milliseconds between calls
getTimerInterval() - (int)

getTickCount() - (int) Returns total number of ticks since the last startTimer() call 

resetTickCount() - (int) Resets the tick count to 0
